# Todo List App

A modern, interactive Todo List App built with HTML, CSS, and JavaScript.
Users can add, edit, complete, and delete tasks, with real-time counters
for total and completed tasks. The app features a pink/peach/coral
gradient UI that stands out on dark backgrounds.

---

## Features

- Add, edit, complete, and delete tasks
- Clear all tasks with one click
- Real-time task counters
- Responsive and visually appealing design

---

## Installation

1. Clone the repository:
   git clone <repository-url>
2. Open the folder and launch index.html in a web browser

---

## Usage

1. Type a task in the input field
2. Click Add to insert it into the list
3. Use edit, complete, or delete buttons for task management
4. Click Clear All to remove all tasks
5. Task counters update automatically
